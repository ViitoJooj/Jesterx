# Jesterx
Jesterx 是一款**SaaS 页面构建器**。
初始版本 (v1) 将在 GitHub 上开源，既是一个功能完善的产品，也是一个技术实验项目。该项目专注于**代码简洁性**、实践学习和理念验证。
其目标是研究可扩展的架构，并尽可能在未来实现盈利。

### 技术栈及选择
- **Golang** api.jesterx.com.br（后端）
- **React** jesterx.com.br（前端）
- **PostgreSQL**
- **Docker**

---

## 概述
本仓库汇集了启动电子商务业务所需的一切：
- 业务规则 API
- Web 界面
- 数据库结构
我们的目标是让任何人都能轻松创建电子商务业务，无需过多配置。

---

## 功能
- 用户注册和身份验证
- 产品 CRUD（创建、读取、更新、删除）
- 购物车
- 订单系统
- 结账
- 管理后台
- 支付集成
